## 🚀 Getting Started

### 🔧 Installation

1. Clone the Portfolio-XD repository:

```sh
git clone https://github.com/prashsamosa/Portfolio-XD
```

2. Change to the project directory:

```sh
cd Portfolio-XD
```

3. Install the dependencies:

```sh
► npm install
```

### 🤖 Running Portfolio-XD

```sh
► npm run dev
```
